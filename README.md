# Houdini Engine Qt

Qt based parameter view widget for Houdini Engine.

# Requirements
- Qt 4.8 (4.8.5)
- QtCreator 3 (3.2.1)
- Houdini 14

## Environment Variables
#### HOUDINI_ROOT

    set HOUDINI_ROOT=[your houdini install folder]
    example:
    set HOUDINI_ROOT=C:\Program Files\Side Effects Software\Houdini 14.0.230
